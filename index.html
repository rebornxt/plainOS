เข้าสู่ระบบเพื่อซิงค์วัตถุดิบและสูตรขนมระหว่างมือถือ แท็บเล็ต และคอมพิวเตอร์ ข้อมูลในเครื่องยังทำงานได้ตามปกติแม้ออฟไลน์

เข้าสู่ระบบในชื่อ —
ยังไม่เคยซิงค์
วิธีตั้งค่า Firebase (กดเพื่อดู)
  1. สร้างโปรเจกต์ที่ console.firebase.google.com
  2. เปิด Build → Firestore Database → Create database
  3. เปิด Build → Authentication → Sign-in method → Email/Password → Enable
  4. ไปที่ Project settings → Your apps → Web (</>) คัดลอกค่า firebaseConfig
  5. วาง config ลงในไฟล์ index.html ตรงบล็อก window.FIREBASE_CONFIG (ในส่วน <head>)
  6. ตั้ง Security Rules ของ Firestore เป็น:
rules_version = '2';
service cloud.firestore {
  match /databases/{database}/documents {
    match /bakery_syncs/{uid} {
      allow read, write: if request.auth != null
                         && request.auth.uid == uid;
    }
  }
}
